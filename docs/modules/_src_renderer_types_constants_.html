Module "src/renderer/types/constants"

constants.d.ts

This file contains definitions related to the redux actions.

The segment of action type constants (defined by being in all caps) define the strings used as action types as constants, to ease the use of type fitting in the reducers.

The segment of action interface definitions define actions to be read by the action creators and consumed by the reducers.
